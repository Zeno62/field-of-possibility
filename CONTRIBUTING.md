# Contributing to Field of Possibility

Thank you for your interest in contributing to **Field of Possibility / 可能性网络**.

This project welcomes discussion, questions, corrections, translations, and thoughtful proposals.

However, this is not a project for creating a new belief system.

All contributions should serve one purpose:

> To help readers see the mechanism of mistaking the apparent for the real, rather than creating a new object of belief.

## Ways to Contribute

You can contribute in the following ways:

- Report typos or formatting issues.
- Suggest clearer wording.
- Ask questions about concepts.
- Propose terminology improvements.
- Suggest translation corrections.
- Discuss chapter structure.
- Submit reading notes.
- Propose new application questions.
- Help review multilingual editions.

## What We Welcome

We welcome contributions that:

- improve clarity;
- preserve conceptual precision;
- help readers understand the framework;
- identify logical ambiguity;
- improve translation accuracy;
- correct typos, formatting, or broken links;
- respectfully question unclear ideas;
- help prevent misunderstanding.

---

## What We Avoid

Please avoid contributions that:

- turn the framework into a religion;
- turn the framework into self-help or success teaching;
- use quantum physics as mystical proof;
- introduce unsupported scientific claims;
- over-personalize the project around one individual;
- dilute core terms without discussion;
- rewrite key concepts without explaining why;
- create a new belief object instead of clarifying observation.

## Translation Contributions

When contributing translations, please keep the following in mind:

- Do not translate key terms too freely.
- Follow the project glossary when available.
- If a term is difficult to translate, open a discussion first.
- Keep chapter numbers aligned with the Chinese source edition.
- Mark translation status clearly: draft, reviewed, official, or outdated.

## Core Boundary

This project is open, but not conceptually shapeless.

The central aim is to observe:

- how consciousness arises;
- how identity forms;
- how suffering repeats;
- how weights shape fate-like patterns;
- how the apparent is mistaken for the real;
- how single-frame interruption becomes possible;
- how return is recognition, not movement toward a place.

Please contribute in a way that protects this clarity.
