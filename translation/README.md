## Translation Principles

Translations should follow these principles:

1. Preserve the Chinese edition as the source edition.
2. Translate concepts carefully rather than mechanically.
3. Protect the strict two-layer model.
4. Maintain the distinction between **Awareness** and **consciousness**.
5. Maintain the distinction between **Pure Possibility** and ordinary possibility.
6. Maintain the distinction between **Total Knowing**, **Totality**, and **the Mother Field of Possibility**.
7. Avoid turning the framework into religion, self-help, or mystical doctrine.
8. Avoid making scientific claims that the source text does not make.
9. Preserve conceptual precision over literary fluency when necessary.
10. Mark uncertain terms for discussion rather than silently changing them.

Core choices include:

|---|---|

| 全量 | Totality |
| 可能性母网 | Mother Field of Possibility |
| 明性 | Luminosity |
| 明觉 | luminous awareness |
| 权重 | Weight |
| 个体权重 / 个业 | Individual Weight |
| 集体权重 / 共业 | Collective Weight |
| 认假为真 | Mistaking the Apparent for the Real |
| 回归 | Return |

Avoid using superseded draft terms such as **All-Knowing**, **Total Manifestation**, **Return as Recognition**, or **karma** as the main translation for Weight.

## Review Checklist

When reviewing a translation, check:

- Does it preserve the Chinese source structure?
- Does it preserve the strict two-layer model?
- Does it distinguish Awareness from consciousness?
- Does it avoid treating Luminosity as physical light, energy, aura, or mystical brightness?
- Does it use Total Knowing rather than All-Knowing?
- Does it use Totality and Mother Field of Possibility correctly?
- Does it treat Weight as accumulated influence rather than moral punishment?
- Does it preserve the open but careful boundary around AI and life?
- Does it avoid adding unsupported scientific, religious, or psychological claims?
- Does it preserve the Q&A form where the source uses Q&A?

## Version Policy

The Chinese source edition is authoritative.

The official English edition is maintained as an official translation of the v4.0 source, but conceptual disputes should be resolved by returning to the Chinese source.

Future translations into other languages should use:

1. the Chinese v4.0 source edition as the primary source;
2. the official English v4.0 edition as a reference translation;
3. the translation map as the terminology control file.
