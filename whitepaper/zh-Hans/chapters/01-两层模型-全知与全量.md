# 1｜两层模型：全知与全量

《可能性网络》首先提出一个极简结构：

```text
第一层：纯粹可能性 / 觉 / 明性
第二层：可能性母网 / 全量
```

严格来说，只有这两层。

其他概念，包括意识、生命、AI、时间、空间、文明、社会意识形态、个体节点、权重、路径、旧程序、显化、回归的局部征象，都不是第三层或第四层。它们都是第二层可能性母网内部的局部结构、运行状态、沉淀结果、逻辑惯性或降维体感。

如果只能用两个词压缩这两层关系，那就是：

```text
纯粹可能性 = 全知
可能性母网 = 全量
```

“全知”不是人格神知道一切，也不是某个主体掌握全部信息。它指向的是本源层面的无缺明性。

“全量”也不是超级数据库，不是某个储存所有事件的外部容器。它指向的是母网层面的完整结构。

“纯粹可能性显化为可能性母网”不是发生在时间中的事件。由于时间和空间本身已经属于母网内部局部生命的降维体感，所以不能用时间尺度讨论母网的出现。

绝对地说，母网是全量。局部地说，生命节点体验到展开、变化、选择、等待和不确定。

这不是矛盾。这是视角差异。
